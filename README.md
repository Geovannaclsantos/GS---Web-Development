## Sobre o Projeto
Este projeto apresenta um formulário interativo que simula a inscrição de um participante em atividades ligadas ao desenvolvimento profissional no agronegócio.

O formulário permite:

- Inserir dados pessoais  
- Escolher habilidades da lista  
- Armazenar habilidades em um vetor  
- Validar dados digitados  
- Exibir erros de forma organizada  
- Mostrar um resumo completo do cadastro quando válido  

---

## Funcionalidades Implementadas

### Validação completa do formulário
- Nome com no mínimo 5 caracteres  
- CPF com 11 dígitos  
- E-mail em formato válido  
- Pelo menos 3 habilidades adicionadas  
- Tipo de interesse obrigatório  

### Uso de JavaScript para:
- Manipulação de DOM  
- Manipulação de Strings  
- Vetores (armazenamento de habilidades)  
- Eventos com addEventListener  
- Atualização dinâmica da interface  

### Funções obrigatórias no projeto:
- adicionarHabilidade()  
- checarEmail()  
- checarCPF()  
- exibirFeedback()  
- validarFormulario()
